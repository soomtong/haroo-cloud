haroo-cloud-core
================

cloud service stack for haroopad

> rest api end point for haroopad

## Features

- account management
- simple review document view for own account
- trend and featured contents

## Service Stack

### backend

- couchdb
- mongodb
- nginx
- node.js
- express.js

### frontend

- jquery
- react.js
- pure
